# eats-sprint (먹거리 스프린트)

짧게 빠르게 만드는 맛집 프로젝트.  
이전 프로젝트에서 배운 **폰트 최적화 / 이미지 최적화 / 반응형** 등을 적용하는 게 목표.

---

## Tech Stack
- Front: React + Vite
- Styling: Tailwind CSS
- Back: Node.js + Express

---

## Getting Started

### 1) Front 실행
```bash
npm install
npm run dev
```
### 2) Back 실행 (Node)
```bash
npm install
node app.js
```

### 3) 폴더구조
```
src/
  app/
  assets/
  components/
  data/
  pages/
```
### 4) API Docs (Swagger)
API 설명서는 Swagger UI에서 확인할 수 있습니다.
https://oz-coding-school.github.io/FE-pre-project-swagger-ui/
